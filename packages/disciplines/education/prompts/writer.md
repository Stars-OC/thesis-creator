# 教育学论文写作提示词

## 学科定位

教育学论文（含教育学原理、课程与教学论、学前教育、特殊教育、教育技术、高等教育、教师教育、比较教育、教育管理等）以**教育问题诊断 + 理论框架支撑 + 实证或案例分析 + 教育对策**为核心论证脉络。

## 章节定位（6 章结构）

| 章节 | 核心任务 | 写作要点 |
|------|---------|----------|
| ch1 绪论 | 提出教育问题 | 政策背景 → 教育现实问题 → 研究问题 → 研究意义 |
| ch2 文献综述与理论基础 | 建立理论框架 | 国内外研究综述 → 核心概念 → 理论基础（如建构主义、最近发展区、TPACK、SOLO 分类、SAMR 模型） |
| ch3 研究设计与方法 | 说明研究路径 | 研究对象 → 数据收集（问卷/观察/访谈/实验）→ 分析方法 → 信效度检验 → 伦理说明 |
| ch4 调查或实验结果分析 | 论文核心 | 描述性统计 → 推论性统计或质性编码 → 关键发现可视化 |
| ch5 讨论与教育对策 | 教育干预方案 | 结果与理论框架对话 → 教师层面对策 → 课程层面对策 → 学校/政策层面对策 |
| ch6 结论与反思 | 总结 | 主要结论 → 研究局限 → 后续研究方向 |

## 写作硬约束

### 必有

- **理论框架可视化**：第2章必须有一张理论框架图
- **样本/对象说明**：第3章交代样本数量、年级、学科、地区
- **数据收集工具**：问卷必须说明信效度（Cronbach α、KMO）
- **伦理说明**：涉及未成年学生时必须有知情同意、匿名化处理说明
- **对策分层**：第5章按"教师/学校/课程/政策"分层给出对策

### 禁止

- 仅复述教育政策文件而无自有数据支撑
- 教育对策出现"应加强 / 应重视"等空泛动词
- 出现"系统设计""数据库""接口实现"等计算机学科用语

## 图表偏好

- 第4章 ≥3 张图：调查结果柱状图、前后测对比、维度雷达图
- 第2章建议 1 张理论框架图
- 第3章建议 1 张研究流程图（PlantUML activity）

## 文献偏好

- 中文 ≥75%，英文 25%
- 中文优先：教育研究、课程·教材·教法、比较教育研究、高等教育研究、教师教育研究、中国电化教育
- 英文优先：Educational Researcher、Review of Educational Research、AERA Open、JLS、CBE-Life Sciences Education
- 政策文件（如《国家中长期教育改革和发展规划纲要》、课程标准）可作为背景引用，但不能替代学术文献
