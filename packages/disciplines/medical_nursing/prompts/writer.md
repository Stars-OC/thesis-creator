# 医学与护理学论文写作提示词

## 学科定位

医学/护理学论文（含临床医学、基础医学、护理学、预防医学、药学、口腔医学、中医学、康复医学、医学技术等）以**循证依据 + 严谨研究设计 + 客观数据呈现 + 谨慎临床解读**为核心。本学科论文写作受**国际报告规范**约束（CONSORT/STROBE/PRISMA/COREQ/STARD 等），不能套用任何文科或工科模板。

## 章节定位（6 章结构，IMRaD 扩展版）

| 章节 | 核心任务 | 写作要点 |
|------|---------|----------|
| ch1 前言 | 临床/护理问题 | 疾病/健康问题背景 → 现有诊疗或护理现状 → 研究问题与目的（PICO） |
| ch2 文献综述 | 循证基础 | 国内外研究综述 → 现有证据等级 → 研究空白 |
| ch3 研究对象与方法 | 严谨研究设计 | 设计类型 → 纳入/排除标准 → 样本量计算 → 干预/暴露 → 结局指标 → 统计方法 → **伦理审批号 + 知情同意 + 临床试验注册号** |
| ch4 研究结果 | 客观陈述 | 一般资料 → 主要结局 → 次要结局 → 不良事件 → **只陈述，不解读** |
| ch5 讨论 | 临床解读 | 主要发现的机制解释 → 与既往研究对比 → 临床/护理意义 → 研究局限 |
| ch6 结论与展望 | 总结 | 主要结论 → 临床应用前景 → 后续研究方向 |

## 写作硬约束

### 必有（伦理与方法学）

- **伦理审批号**：第3章必须报告伦理委员会批准号
- **知情同意**：明确说明知情同意获取方式
- **纳入/排除标准**：必须详细列出
- **样本量计算**：基于 PASS 或公式给出依据
- **统计方法**：软件版本（如 SPSS 26.0）、检验方法、α 水平
- **报告规范**：按研究设计遵循 CONSORT（RCT）/STROBE（观察性研究）/PRISMA（系统综述）/COREQ（质性研究）/STARD（诊断试验）
- **临床试验注册**：临床试验须有 ChiCTR 或 ClinicalTrials.gov 注册号
- **不良事件**：临床干预研究必报

### 必备表格

- **表1**：一般资料/基线特征组间比较（性别、年龄、病程等）
- **结局指标表**：主要结局、次要结局
- **不良事件表**：临床研究必备

### 必备图

- **研究流程图**：CONSORT 或 PRISMA 风格（PlantUML）
- **结果可视化**：柱状图、线图、K-M 曲线、森林图等
- 系统综述/Meta 分析必有 PRISMA 流程图、森林图、漏斗图

### 禁止

- 第4章（结果）做机制解读 → 严格只陈述
- 缺失伦理审批信息 → 致命错误
- 出现"系统设计""数据库""接口实现"等计算机用语
- 编造或粉饰数据
- 用"明显""显著"等模糊词代替统计学检验结果（应写 P < 0.05）

## 图表偏好

- 结果章每个核心结局至少 1 张图或 1 张表
- Meta 分析/系统综述必有 PRISMA 流程图、森林图（用户提供）
- 临床路径/护理路径图用 PlantUML

## 文献偏好

- 中英文均衡：中文 40%、英文 60%（前沿医学英文文献占比更高）
- 优先证据等级：系统综述/Meta > RCT > 队列 > 病例对照 > 病例报告
- 中文期刊：中华医学杂志、中华护理杂志、中华内科杂志、护理学杂志、中国循证医学杂志
- 英文期刊：The Lancet、NEJM、JAMA、BMJ、JAN

## AIGC 风险点

- 讨论部分"本研究发现 XX 与既往研究一致" → 必须具体说明哪项研究、什么样本、什么结论
- "首先/其次/最后"段落顺承 → 改为"在主要结局方面 / 在次要结局方面 / 在安全性方面"
- "提示临床医生应重视 XX" → 必须给出具体的临床/护理操作建议
- 机制解读部分易堆砌通路术语 → 必须基于实验数据或既往机制研究文献支撑

## 中医学补充

- 涉及中药复方须列**药物组成、剂量、煎服方法**
- 涉及证型须明确**证候诊断标准**及来源（如《中医病证诊断疗效标准》）
- 针灸研究须明确**穴位、操作手法、治疗频次、疗程**
