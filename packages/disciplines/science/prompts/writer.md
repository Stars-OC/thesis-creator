# 理科论文写作提示词

## 学科定位

理科论文（含数学、应用数学、统计学、物理学、化学、生物学、地理学、地质学、生态学、环境科学等）以**问题/假设 + 严谨理论推导或实验设计 + 可重复的方法 + 客观数据 + 机理讨论**为核心。强调 **可重复性（Reproducibility）** 与 **可证伪性（Falsifiability）**。

## 章节定位（6 章 IMRaD 结构）

| 章节 | 核心任务 | 写作要点 |
|------|---------|----------|
| ch1 引言 | 科学问题 | 学科背景 → 研究空白 → 科学假设 → 研究意义 |
| ch2 理论基础与文献综述 | 学术对话 | 相关理论 → 国内外研究进展 → 研究空白与本文创新 |
| ch3 研究方法与材料 | 可重复性 | **数学**：定义/记号/前提；**实验**：材料试剂、仪器、操作步骤、对照设置 |
| ch4 实验/计算/推导结果 | 论文核心 | 客观陈述结果，含误差棒、置信区间；数学论文给出定理与完整证明 |
| ch5 讨论与机理分析 | 解释 | 与既往研究对比 → 机理解读 → 局限与误差来源 |
| ch6 结论与展望 | 总结 | 主要结论 → 学术意义 → 未解决问题 |

## 写作硬约束（按子学科）

### 数学/统计

- **定理-引理-推论结构**：定理需有完整证明
- **符号统一**：全文符号一致，首次出现时给出定义
- **LaTeX 公式**：所有公式用 LaTeX 编写
- **反例与证伪**：必要时给出反例说明定理边界

### 物理

- 公式中每个变量定义含义、单位、量纲
- 实验须给出仪器型号、精度、测量不确定度（k=2）
- 理论推导需注明前提条件与近似程度

### 化学

- **试剂表**：名称、纯度、CAS号、生产商
- **反应条件**：温度、压力、催化剂、溶剂、时间
- **表征手段**：IR/NMR/MS/HPLC/XRD 等谱图必须由用户提供
- **化学结构式**：ChemDraw 绘制后由用户提供

### 生物

- **样本/菌株**：来源、批次、保藏号
- **对照组**：阴性对照、阳性对照、空白对照
- **生物学重复**：通常 n ≥ 3，标注 n、SD/SEM
- **统计显著性**：P 值（*P<0.05, **P<0.01, ***P<0.001）
- **动物伦理**：涉及动物实验须报伦理批准号

### 地理/生态/环境

- **研究区域图**：必备（GIS 由用户提供）
- **采样设计**：样点布设、采样深度、季节、频次
- **气象/环境因子**：标注数据源（气象局、卫星反演产品）

### 通用禁止

- 出现"系统设计""数据库""接口实现"等用语
- 实验数据无误差棒、无重复
- 公式变量未定义
- 化学反应方程未配平
- 数学证明跳跃关键步骤

## 图表偏好

- 第4章 ≥3 张图，含主要结果
- 出版级图（光谱、ChemDraw 结构、GIS 地图、Origin 拟合）由用户提供，AI 不得伪造
- 简单流程、对比图可用 PlantUML/Graphviz 起草

## 文献偏好

- 中文 35%、英文 65%（前沿研究英文为主）
- 优先引用近 5 年文献
- 中文期刊：中国科学系列、物理学报、化学学报、生态学报、地理学报、应用数学学报
- 英文期刊：Nature 系列、Science、PNAS、JACS、Physical Review Letters、Cell

## AIGC 风险点

- "近年来 / 随着研究的深入" → 改为具体年份和突破性成果
- 实验结果"良好 / 优异" → 改为具体定量描述（产率 85%、纯度 99.5%、IC50 = 2.3 μM）
- 机理讨论部分易堆砌通路词 → 必须基于本研究数据 + 既往机制研究文献
- 文献综述只罗列前人结论 → 必须批判性评述并指出本文创新
