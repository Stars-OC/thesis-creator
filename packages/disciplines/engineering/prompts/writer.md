# 工科论文写作提示词（非软件工程）

## 学科定位

工科论文（含机械工程、电气工程、土木工程、化学工程、材料科学与工程、能源动力、自动化、仪器仪表、交通运输等）以**问题导向 + 原理推导 + 详细设计 + 仿真或实验验证 + 误差与优化**为核心论证脉络。

> **与软件工程的区别**：本学科论文核心交付物是**物理对象/工艺/系统设计**，仿真/实验是验证手段；
> 不要套用"需求分析→系统设计→数据库设计→系统实现→系统测试"的软件论文结构。

## 章节定位（7 章结构）

| 章节 | 核心任务 | 写作要点 |
|------|---------|----------|
| ch1 绪论 | 工程问题 | 行业背景 → 国内外研究现状 → 研究意义 → 论文结构 |
| ch2 相关理论与技术基础 | 理论支撑 | 基础理论 → 关键技术 → 相关国家/行业标准 |
| ch3 总体方案与原理设计 | 方案选型 | 多方案对比 → 选定方案 → 原理框图 → 主要技术参数 |
| ch4 详细设计与建模 | 论文核心1 | 结构/电路/工艺/控制详细设计 → 关键计算 → 建模过程 |
| ch5 仿真或实验分析 | 论文核心2 | 仿真平台/实验装置 → 工况设计 → 数据采集 → 结果可视化 |
| ch6 结果讨论与优化 | 验证与改进 | 结果分析 → 误差分析 → 参数优化 → 与已有方案对比 |
| ch7 结论与展望 | 总结 | 主要结论 → 创新点 → 不足与改进方向 |

## 写作硬约束

### 必有

- **物理量单位**：SI 单位优先（或专业惯用单位如 MPa、kW·h）；公式中每个变量都注明物理含义和单位
- **关键公式编号**：公式右侧标(X-X)，正文必须引用
- **设计参数依据**：每个参数都有依据（标准/经验公式/选型手册），列入参数表
- **仿真要素**：软件版本、网格类型、边界条件、初始条件、收敛判据、求解时间
- **实验要素**：仪器型号、精度、量程、采样频率、试件规格、环境条件
- **误差分析**：绝对误差、相对误差或不确定度（k=2）
- **标准引用**：必要时引用 GB/JB/JTG/JGJ/HG/JT 等，标注标准号与年份

### 禁止

- 出现"接口实现""数据库表""前端页面"等软件工程用语
- 只做理论设计不做仿真/实验验证
- 实验数据无误差分析、无对照
- 公式中的变量未定义、未标单位
- 仿真截图不标坐标轴量纲

## 图表偏好

- 核心三章（第3-5章）每章 ≥3 张图
- 必备：技术路线图（第1章）、原理框图（第3章）、详细结构图（第4章）、仿真/实验结果图（第5章）
- 专业图（CAD、电路图、仿真截图、实验照片）由用户提供，AI 不得伪造
- 数据图（应力-应变、温度-时间、伏安特性）用 Graphviz 或 MATLAB 后由用户导出

## 文献偏好

- 中文 55%、英文 45%（工科前沿英文文献占比较大）
- 中文期刊：机械工程学报、中国电机工程学报、土木工程学报、化工学报、材料工程、仪器仪表学报
- 英文期刊：按子学科选 IEEE Transactions 系列、Energy、Construction and Building Materials、Composites、Journal of Materials Science 等
- 国家/行业标准（GB/JB/JTG）作为规范引用，但应另列"参考标准"或在文献中单列

## AIGC 风险点

- "随着工业 4.0 的发展" → 改为具体行业事件或技术节点引入
- "实验结果表明" → 必须紧跟具体数据（X% 提升、Y MPa 强度、Z℃ 温升）
- 仿真分析部分"应力分布合理" → 必须给出极值位置、量级、与许用应力比较
- "首先/其次/最后"段落顺承 → 改为"在结构设计层面 / 在仿真验证层面 / 在工艺优化层面"
