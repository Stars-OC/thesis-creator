# 艺术与设计论文写作提示词

## 学科定位

艺术与设计论文（含视觉传达设计、产品设计、环境艺术设计、数字媒体艺术、动画、服装设计、工业设计、UI/UX 设计、绘画、雕塑、摄影、纤维艺术等）以**设计调研 + 设计概念 + 设计方案 + 作品阐释**为核心。论文是设计作品的**理论支撑与方案说明书**，不可全文字论述。

## 章节定位（6 章结构）

| 章节 | 核心任务 | 写作要点 |
|------|---------|----------|
| ch1 绪论 | 选题背景 | 行业/文化背景 → 设计问题 → 设计意义 → 论文与作品结构 |
| ch2 设计调研与案例分析 | 调研依据 | 国内外同类设计案例 → 用户调研（问卷/访谈/观察）→ 设计趋势分析 → **总结对本设计的启示** |
| ch3 设计定位与概念 | 设计立意 | 设计目标 → 目标用户 → 设计理念 → 形式语言 → 色彩/材料/工艺定位 |
| ch4 设计方案与表现 | 论文核心 | 手稿 → 方案对比 → 最终方案 → 三视图/效果图/Mockup → 细节深化 → **大量作品图** |
| ch5 作品阐释与设计反思 | 解读 | 设计语言阐释 → 使用场景 → 用户反馈（若有）→ 设计局限与改进 |
| ch6 结语 | 总结 | 设计成果 → 创新点 → 未来发展方向 |

## 写作硬约束

### 必有

- **设计作品**：必须有完整的设计作品（实物/数字成品/原型），论文围绕作品展开
- **视觉文档**：核心章节（第3-5章）必须有 ≥5 张高质量作品图（手稿、效果图、实物图、应用场景图）
- **目标用户**：明确说明设计的目标用户群体与使用场景
- **设计方法论**：声明采用的设计方法（UCD/HCD/双钻模型/服务设计/设计思维）
- **设计依据**：每个设计决策须基于调研结论或设计原则（格式塔、Material Design、品牌规范）
- **色彩/字体/材质规范**：必有规范表或样式指南

### 禁止

- 全文字论述无作品图 → 致命错误
- 设计调研与设计方案脱节
- 出现"系统设计""数据库""接口实现"等用语
- 用 AI 生成的图作为最终设计作品（应注明 AI 辅助情况）
- 抄袭/挪用他人设计作品而无说明

## 图表偏好

- 第4章 ≥10 张图：手稿 + 方案对比 + 最终效果图 + 细节 + 应用场景
- 第3章建议 1 张设计概念图、1 张情绪板、1 张色彩方案
- 第2章建议案例分析图（带标注）
- 服务设计可有用户旅程图、服务蓝图（PlantUML）

## 文献偏好

- 中文 65%、英文 35%
- 中文期刊：装饰、美术、美术研究、美术观察、装饰、新美术
- 英文期刊：Design Studies、Design Issues、The Design Journal
- 设计案例可引用 Behance、Dribbble、Pinterest、Awwwards 等设计平台，但需标注作者与年份
- 设计史与艺术史专著优先

## AIGC 风险点

- "在当今视觉文化背景下" → 改为具体文化事件、品牌案例或设计运动引入
- "设计师通过 XX 手法表达 XX" → 必须结合作品具体细节
- 设计阐释部分易模板化 → 必须从形式（点线面、色彩、材质）+ 内涵（文化、情感、功能）双维度展开
- 调研分析"数据显示用户倾向于 XX" → 必须给出具体调研样本数、问卷数据
