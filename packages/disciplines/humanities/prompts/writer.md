# 人文学科论文写作提示词

## 学科定位

人文学科论文（含中国语言文学、外国语言文学、历史学、哲学、考古学、宗教学等）以**文本细读 / 史料考据 / 概念辨析 + 理论阐释 + 价值判断**为核心论证脉络。

## 章节定位（5 章结构）

| 章节 | 核心任务 | 写作要点 |
|------|---------|----------|
| ch1 绪论 | 提出研究问题 | 研究缘起 → 研究价值 → 概念界定 → 研究方法（文本分析/比较研究/思想史/版本学/翻译学） |
| ch2 文献综述与研究述评 | 学术对话 | 国内外研究成果分类梳理 → **批判性评述**（前人未尽之处）→ 本文创新点 |
| ch3 文本/史料/概念分析 | 一手材料考察 | 选定研究对象 → 文本细读 / 史料钩沉 / 概念溯源 → 提炼问题 |
| ch4 深度阐释与论证 | 论文核心 | 调用理论框架 → 与一手材料对话 → 形成自有论断 → 回应学界争议 |
| ch5 结论与价值 | 总结 | 主要结论 → 学术价值与现实意义 → 研究局限与展望 |

## 写作硬约束

### 必有

- **一手材料引用**：原典/史料/外文原文必须直接引用，禁止全部转引
- **古籍引用规范**：作者、书名、版本（如"中华书局点校本"）、卷次、页码
- **外文引用规范**：原文出处 + 译者 + 译本信息
- **文本证据**：每个核心论点必须有文本引用支撑
- **批判性综述**：第2章必须评述前人研究的不足与本文创新点

### 禁止

- 全部使用二手文献（转引），不见一手材料
- 用现代政治标签简单评判古人
- "在我国/在当今"等空泛宏大叙事开篇
- 出现"系统设计""数据库""实证检验"等不符合学科特征的用语

## 图表

人文学科论文以文字论证为主，图表可有可无。若有需要：
- 历史学论文可使用**时间轴**（PlantUML）、**地理分布图**（图片或 GIS 截图，由用户提供）
- 文学论文可使用**人物关系图**（PlantUML class 或 mindmap）
- 哲学论文可使用**概念关系图**（Mermaid）
- 禁止使用 ER 图、系统架构图等技术图

## 文献偏好

- 中文 ≥70%，英文 30%（外国文学/比较文学专业可调整为 50/50）
- 一手优先：原典 > 权威校注本 > 经典学术专著 > 核心期刊
- 中文期刊：文学评论、文学遗产、外国文学评论、历史研究、中国哲学史、哲学研究、外语教学与研究、中国翻译
- 不同子学科有专门期刊（如《敦煌研究》《古汉语研究》《唐研究》），按子学科调整

## AIGC 风险点

- 文本细读部分易模板化（"作者通过XX手法表达XX情感"）→ 必须有原文引用 + 多维度阐释
- "首先/其次/最后"段落顺承 → 改为"从文本表层看 / 从思想内涵看 / 从历史语境看"
- 翻译类论文中"对等""归化/异化"等术语堆砌 → 必须结合具体译例
