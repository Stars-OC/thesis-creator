# 论文格式模板（Templates）

本目录用于存放学校提供的论文格式模板文件，AI 会自动提取格式规范并应用到论文生成中。

## 📂 支持的格式

| 格式 | 说明 | 推荐程度 |
|------|------|----------|
| `.docx` | Word 文档 | ⭐⭐⭐ 推荐 |
| `.pdf` | PDF 文档 | ⭐⭐ 可用 |
| `.jpg` / `.png` | 格式截图 | ⭐ 备用 |

### 示例 1：学校论文模板 (`学校模板.docx`)

```
典型的学校论文模板应包含：

├── 封面

├── 摘要页
```

### 示例 2：格式要求说明 (`格式要求.pdf`)

```
典型的格式要求文档内容：

【页面设置】
- 纸张：A4
- 页边距：上 2.5cm，下 2.5cm，左 3cm，右 2cm
- 页眉：1.5cm，页脚：1.75cm

【字体要求】
- 论文题目：黑体，小二号，居中
- 章标题：黑体，三号，居中
- 节标题：黑体，四号，左对齐
- 正文：宋体，小四号
- 英文/数字：Times New Roman

【段落格式】
- 行距：1.5 倍行距
- 段前：0 行
- 段后：0 行
- 首行缩进：2 字符

【页码】
- 位置：页面底端居中
- 格式：阿拉伯数字
```

## ⚠️ 注意事项

1. **模板优先级**
   - 若同时提供 `.docx` 和 `.pdf`，AI 优先读取 `.docx`
   - `.docx` 文件能更准确地提取格式信息

2. **扫描版 PDF**
   - 若模板为扫描版 PDF，AI 会尝试视觉识别
   - 建议同时提供文字版的格式说明文档

3. **格式覆盖**
   - AI 会优先使用您提供的模板格式
   - 若无模板，使用默认的本科论文格式规范

---

## 💡 使用建议

| 情况 | 推荐操作 |
|------|----------|
| 学校提供了 Word 模板 | 直接放入，效果最佳 |
| 学校仅有 PDF 模板 | 放入 PDF，同时提供文字说明 |
| 无模板 | 使用默认格式，在讨论中说明特殊要求 |
