# 参考资料（Reference）

本目录用于存放您的研究资料，**完全由您自定义组织**。AI 会读取此目录下的所有文件。

---

## 💡 使用方式

**自由发挥，想放什么放什么。**

```
reference/
├── 你的代码.py
├── 你的论文.pdf
├── 你的笔记.txt
├── 你的数据.xlsx
└── ... 任何你觉得有用的文件
```

也可以创建子目录整理：

```
reference/
├── 代码/
│   ├── data_process.py
│   └── model_train.py
├── 论文/
│   ├── 相关论文1.pdf
│   └── 相关论文2.pdf
└── 笔记/
    └── 研究笔记.md
```

---

## 📂 支持的格式

AI 可以读取以下格式：

| 类型 | 格式 |
|------|------|
| 代码 | `.py`, `.R`, `.sql`, `.ipynb`, `.java`, `.js`, `.ts`, `.go`, `.vue` 等 |
| 文档 | `.pdf`, `.docx`, `.md`, `.txt` |
| 数据 | `.csv`, `.json`, `.xlsx`, `.xml` |

---

## ⚠️ 注意事项

1. **AI 全量读取**：此目录下所有非 README 文件都会被读取
2. **排除文件**：`README.md` 会被自动排除，不会被当作内容读取
3. **敏感信息**：请勿放入密码、密钥、个人隐私数据

---

## 📖 子目录说明（可选）

如果您想使用预定义的子目录结构，可以参考：

- `code/` - 存放代码相关文件
- `doc/` - 存放文档相关文件

这两个子目录有对应的 README 说明，但**您可以完全忽略，按自己的方式组织**。
