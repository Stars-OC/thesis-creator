# AIGC 降低提示词指南

> 本文档用于指导 AI 模拟人类写作特征，降低 AIGC 检测系统的识别概率

---

## 一、AIGC 检测原理

### 1.1 检测维度

主流 AIGC 检测系统关注以下特征：

| 检测维度 | AI 文本特征 | 人类文本特征 |
|---------|------------|-------------|
| **困惑度（Perplexity）** | 低（语言模型可预测） | 高（表达多样不可预测） |
| **突发性（Burstiness）** | 低（句长均匀） | 高（句长波动大） |
| **词汇多样性（TTR）** | 低（词汇重复率高） | 高（词汇丰富） |
| **过渡词密度** | 高（模板化过渡） | 低（自然过渡） |
| **句式模式** | 整齐（偏好总分总） | 多样（多种结构交替） |

### 1.2 AI 高频特征词汇

以下词汇在 AI 生成文本中出现频率异常高：

| 类别 | 高频词 |
|------|-------|
| 过渡词 | 首先、其次、最后、此外、另外、同时、综上所述、由此可见 |
| 强调词 | 值得注意的是、需要指出的是、不可否认、显然、明确 |
| 总结词 | 总而言之、概括来说、简而言之、换句话说 |
| 程度词 | 非常、十分、极其、相当、一定程度上 |
| 动词 | 旨在、致力于、表现为、体现为、涉及到 |

## 二、特征消除策略

### 2.1 P0 优先级（必做）

#### 2.1.1 消除模板化过渡词

**问题**：AI 文本大量使用「首先…其次…最后」「此外」「综上所述」等模板化过渡

**解决方案**：

| 原表达 | 改写方案 |
|-------|---------|
| 首先，A；其次，B；最后，C | A。B。C。（直接陈述，无过渡词） |
| 此外，研究还发现… | 研究同时发现… / 另一发现是… |
| 综上所述，可以得出以下结论 | 上述分析表明 / 概括以上内容 |
| 值得注意的是 | 值得关注的是 / 需要说明的是 |
| 由此可见 | 这意味着 / 这说明 |

**示例**：

**原文（AI 特征明显）**：
> 首先，大数据技术为精准营销提供了数据基础。其次，大数据技术实现了消费者画像的精准构建。最后，大数据技术支撑了营销决策的优化。

**改写（消除模板）**：
> 大数据技术对精准营销的支撑体现在三个层面：数据基础的奠定、消费者画像的构建、营销决策的优化。

#### 2.1.2 增加句长波动

**问题**：AI 文本句长均匀（多为 20-30 字），人类写作句长波动大（10-50 字）

**解决方案**：刻意制造句长差异

| 操作 | 示例 |
|------|-----|
| 短句强调 | 数据是核心。（6字） |
| 长句阐述 | 在市场竞争日趋激烈、消费者需求日益多元化的背景下，精准营销逐渐成为企业获取竞争优势的关键策略。（45字） |
| 穿插使用 | 市场在变。企业必须适应变化才能生存。而在适应过程中，数据驱动决策的理念逐渐深入人心，成为数字化转型的重要支撑。 |

**目标**：句长标准差 > 10

#### 2.1.3 段落结构多样化

**问题**：AI 偏好「总分总」结构，段落开头千篇一律

**解决方案**：

| 结构类型 | 示例开头 |
|---------|---------|
| 问题引入式 | 「为什么精准营销越来越重要？」 |
| 数据引入式 | 「2023 年市场规模达到 500 亿元。」 |
| 案例引入式 | 「以某电商平台为例。」 |
| 引用引入式 | 「张三（2023）指出：『…』」 |
| 对比引入式 | 「传统营销与精准营销的区别在于…」 |
| 设问引入式 | 「消费者真正想要的是什么？」 |

### 2.2 P1 优先级（建议）

#### 2.2.1 增加主观性表达

**问题**：AI 文本客观性过强，缺乏研究者主观视角

**解决方案**：适度添加主观表达

| 类型 | 表达示例 |
|------|---------|
| 第一人称 | 笔者、本文作者、研究者 |
| 观点表达 | 笔者认为、据笔者观察、在笔者看来 |
| 推测语气 | 据推测、可能意味着、或许可以认为 |
| 质疑语气 | 是否意味着…？值得进一步探讨 |

**示例**：

**原文**：
> 大数据技术的应用显著提高了营销效率。

**改写**：
> 笔者观察发现，大数据技术的应用在提高营销效率方面的效果较为显著。

#### 2.2.2 增加逻辑不完美感

**问题**：AI 逻辑过于完美、线性，缺乏人类思维的跳跃性

**解决方案**：适度添加转折、让步、质疑

| 操作 | 示例 |
|------|-----|
| 添加转折 | 「虽然效果显著，但也存在一定局限。」 |
| 添加让步 | 「当然，这一结论还需更多实证支持。」 |
| 添加质疑 | 「这一现象背后的原因是什么？值得深入思考。」 |
| 承认局限 | 「限于数据可得性，本研究未能覆盖更多样本。」 |

#### 2.2.3 替换 AI 高频词汇

**高频词替换表**：

| AI 高频词 | 替换选项 |
|----------|---------|
| 旨在 | 目标是、目的是、意在 |
| 致力于 | 专注于、着力推进、持续投入 |
| 具有重要意义 | 意义重大、作用显著、价值突出 |
| 表现为 | 具体体现为、呈现出、显现为 |
| 涉及到 | 涵盖、包含、关系到 |
| 一定程度上 | 某种意义上、就部分而言 |
| 显然 | 明显、不难看出、可以预见 |

### 2.3 P2 优先级（可选）

#### 2.3.1 变换段首表达

**问题**：AI 文本每段开头同质化严重

**解决方案**：记录并避免重复的段首模式

| 避免模式 | 替代方案 |
|---------|---------|
| 「XX是…」（定义式开头） | 「对于XX，学界有不同的界定。」 |
| 「随着…」（背景式开头） | 「当前，…已成为共识。」 |
| 「在…背景下」（背景式开头） | 直接切入主题 |

#### 2.3.2 增加标点多样性

**问题**：AI 文本很少使用分号、破折号

**解决方案**：

| 标点 | 使用场景 | 示例 |
|------|---------|-----|
| 分号 | 并列列举 | 数据来源有三个：内部数据；外部数据；第三方数据。 |
| 破折号 | 补充说明 | 精准营销——即基于数据分析的营销方式——正在改变传统营销格局。 |
| 问号 | 提出问题 | 消费者真正需要的是什么？ |

## 三、自检循环

### 3.1 检测流程

```
输入文本
    ↓
调用 aigc_detect.py 检测
    ↓
AIGC 率 ≤ 阈值（默认 15%）?
    ↓
├─ 是 → 通过，进入下一环节
└─ 否 → 标记高风险段落 → 重新改写
         ↓
    最多 3 轮
         ↓
    仍未通过 → 输出最佳版本 + 风险标记
```

### 3.2 高风险段落标记

检测脚本会输出高风险段落编号：

```json
{
  "overall_score": 28.0,
  "high_risk_paragraphs": [2, 5, 8],
  "suggestion": "建议重点改写第 2、5、8 段"
}
```

**改写重点**：
- 第 2 段：检测到「首先…其次…」模板化过渡
- 第 5 段：句长方差过小，缺乏波动
- 第 8 段：过渡词密度过高

### 3.3 CLI 检测命令

```powershell
# 检测单个文件
python scripts/aigc_detect.py --input paper.md

# 检测一段文本
python scripts/aigc_detect.py --text "待检测的文本内容..."

# 指定检测模式
python scripts/aigc_detect.py --input paper.md --mode lite   # 轻量版
python scripts/aigc_detect.py --input paper.md --mode full   # 完整版（含困惑度）
```

## 四、人性化改写示例

### 4.1 完整改写案例

**原文（AI 特征明显）**：

> 此外，值得注意的是，大数据技术在精准营销中的应用具有重要的意义。首先，大数据技术能够实现消费者画像的精准构建。其次，大数据技术可以支撑营销决策的优化。最后，大数据技术有助于提升营销效果。综上所述，大数据技术是精准营销的核心支撑。

**问题分析**：
1. ❌ 「此外」「值得注意的是」— 模板化过渡
2. ❌ 「具有重要的意义」— AI 高频词汇
3. ❌ 「首先…其次…最后」— 模板化列举
4. ❌ 「综上所述」— AI 总结模板
5. ❌ 句长均匀（约 20 字）
6. ❌ 总分总结构

**改写后（人性化）**：

> 大数据技术对精准营销的价值已得到学界广泛认可。笔者在研究中发现，这种价值主要体现在三个层面。消费者画像的构建依赖数据。通过对用户行为数据的采集、整合与分析，企业得以勾勒出目标客户的完整面貌。营销决策的优化同样离不开数据支撑——基于数据分析的决策更加精准、高效。至于营销效果的提升，则是数据价值的直接体现。据张三（2023）的研究，应用大数据技术的企业营销 ROI 平均提升 2.3 倍。

**改写要点**：
1. ✅ 消除「此外」「值得注意的是」
2. ✅ 替换「具有重要的意义」
3. ✅ 消除「首先…其次…最后」
4. ✅ 消除「综上所述」
5. ✅ 句长有明显波动（6字 → 45字）
6. ✅ 多种结构交替（陈述 → 定义 → 数据 → 引用）

## 五、质量自检清单

完成人性化改写后，对照以下清单自查：

- [ ] 无「首先…其次…最后」模板
- [ ] 无「此外」「综上所述」「值得注意的是」等 AI 高频过渡词
- [ ] 句长有明显波动（标准差 > 10）
- [ ] 段落开头形式多样
- [ ] 有适度的主观表达
- [ ] 有转折、让步等不完美感
- [ ] 标点使用多样（有分号、破折号）
- [ ] AIGC 检测率 ≤ 阈值

## 六、注意事项

### 6.1 改写边界

- **保持语义不变**：改写不能改变原文核心观点
- **保护专业术语**：术语不在改写范围内
- **保持学术规范**：人性化不等于口语化

### 6.2 风险提示

- 本地检测为近似估计（准确率 55-80%）
- 正式提交前建议到知网/维普做正式检测
- 不同检测平台算法有差异，结果可能不同

### 6.3 伦理说明

- 本工具用于辅助写作，不是替代写作
- 用户应对最终内容负责
- 建议在改写基础上进行人工审核和修改
