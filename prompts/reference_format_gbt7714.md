# GB/T 7714-2015 参考文献格式规范

> 本文档严格参照 GB/T 7714-2015《信息与文献 参考文献著录规则》编写
> 每次编写论文时必须调用此规范

---

## 一、文献类型标识

| 标识代码 | 文献类型 | 说明 |
|---------|---------|------|
| M | 专著 | 普通图书 |
| J | 期刊文章 | 学术期刊论文 |
| D | 学位论文 | 硕士/博士论文 |
| C | 会议论文 | 会议录、论文集 |
| N | 报纸文章 | 报纸 |
| R | 报告 | 科技报告 |
| S | 标准 | 国际/国家标准 |
| P | 专利 | 专利文献 |
| DB | 数据库 | 在线数据库 |
| CP | 计算机程序 | 软件 |
| EB | 电子公告 | 网页、博客等 |
| Z | 其他 | 未定义类型 |

**组合类型**：
- DB/OL — 在线数据库
- DB/MT — 磁带数据库
- M/CD — 光盘图书
- CP/DK — 磁盘软件
- J/OL — 在线期刊文章
- EB/OL — 在线网页

## 二、著录格式详解

### 2.1 专著(M)

**基本格式**：
```
[序号] 主要责任者. 题名: 其他题名信息[M]. 其他责任者. 版本项. 出版地: 出版者, 出版年: 引文页码.
```

**示例**：
```
[1] 刘国钧, 陈绍业. 图书馆目录[M]. 北京: 高等教育出版社, 1957: 15-18.
[2] 昂温 G, 昂温 P S. 外国出版史[M]. 陈生铮, 译. 北京: 中国书籍出版社, 1988.
[3] 广西壮族自治区林业厅. 广西自然保护区[M]. 北京: 中国林业出版社, 1993.
[4] 霍斯尼 R K. 谷物科学与工艺学原理[M]. 李庆龙, 译. 2版. 北京: 中国食品出版社, 1989: 15-20.
```

**著录要点**：
1. 主要责任者不超过 3 人时全部著录，超过 3 人时著录前 3 人后加「, 等」
2. 欧美作者姓名采用姓前名后，姓全大写，名缩写
3. 版本项：第1版不著录，其他版本需著录
4. 页码为引用页码，非全书页数

### 2.2 期刊文章(J)

**基本格式**：
```
[序号] 主要责任者. 题名[J]. 刊名, 年, 卷(期): 引文页码.
```

**示例**：
```
[5] 何龄修. 读顾城《南明史》[J]. 中国史研究, 1998, (3): 167-173.
[6] 金显贺, 王昌长, 王忠东, 等. 一种用于在线检测局部放电的数字滤波技术[J]. 清华大学学报(自然科学版), 1993, 33(4): 62-67.
[7] 莫少强. 数字式中文全文文献格式的设计与研究[J/OL]. 情报学报, 1999, 18(4): 1-6[2001-07-08]. http://periodical.wanfangdata.com.cn/periodical/qbxb/qbxb99/qbxb9904/990407.htm.
```

**著录要点**：
1. 刊名完整著录，不缩写
2. 年、卷、期齐全时格式为：年, 卷(期): 页码
3. 无卷号时格式为：年, (期): 页码
4. 无期号时格式为：年, 卷: 页码

### 2.3 学位论文(D)

**基本格式**：
```
[序号] 主要责任者. 题名[D]. 保存地: 保存单位, 年份.
```

**示例**：
```
[8] 张志祥. 间断动力系统的随机扰动及其在守恒律方程中的应用[D]. 北京: 北京大学数学学院, 1998.
[9] CALMS R B. Infrared spectroscopic studies on solid oxygen[D]. Berkeley: Univ. of California, 1965.
```

**著录要点**：
1. 保存地为城市名
2. 保存单位为学位授予单位的全称或规范简称
3. 中文学位论文著录中文题名，不著录学位级别

### 2.4 会议论文(C)

**基本格式**：
```
[序号] 主要责任者. 题名[C]//编者. 会议名称, 会议地点, 会议时间: 页码.
```

**示例**：
```
[10] 中国力学学会. 第3届全国实验流体力学学术会议论文集[C]. 天津: [出版者不详], 1990.
[11] ROSENTHALL E M. Proceedings of the Fifth Canadian Mathematical Congress, University of Montreal, 1961[C]. Toronto: University of Toronto Press, 1963.
[12] 张筑生. 微分半动力系统的不变集[D]//北京大学数学系数学研究所. 北京: 北京大学, 1983.
```

### 2.5 科技报告(R)

**基本格式**：
```
[序号] 主要责任者. 报告题名[R]. 报告地: 报告单位, 年份.
```

**示例**：
```
[13] 冯西桥. 核反应堆压力管道与压力容器的LBB分析[R]. 北京: 清华大学核能技术设计研究院, 1997.
[14] U.S. Department of Transportation Federal Highway Administration. Guidelines for handling excavated acid-producing material, PB 91-194001[R]. Springfield: U.S. Department of Commerce National Information Service, 1990.
```

### 2.6 标准(S)

**基本格式**：
```
[序号] 标准编号, 标准名称[S].
```

**示例**：
```
[15] GB/T 7714-2015, 信息与文献 参考文献著录规则[S].
[16] ISO 690-2, Information and documentation—Bibliographic references—Part 2: Electronic documents or parts thereof[S].
```

### 2.7 专利(P)

**基本格式**：
```
[序号] 专利申请者或所有者. 专利题名: 专利号[P]. 公告日期或公开日期.
```

**示例**：
```
[17] 姜锡洲. 一种温热外敷药制备方案: 88105607.3[P]. 1989-07-26.
[18] TACHIBANA R, SHIMIZU S, KOBAYSHI S, et al. Electronic watermarking method and system: US6915001[P/OL]. 2005-07-05[2013-11-11]. http://www.google.com/patents/US6915001.
```

### 2.8 报纸文章(N)

**基本格式**：
```
[序号] 主要责任者. 题名[N]. 报纸名, 出版日期(版次).
```

**示例**：
```
[19] 谢希德. 创造学习的新思路[N]. 人民日报, 1998-12-25(10).
[20] 丁文祥. 数字革命与竞争国际化[N]. 中国青年报, 2000-11-20(15).
```

### 2.9 电子文献(DB/OL, EB/OL, J/OL 等)

**基本格式**：
```
[序号] 主要责任者. 题名[文献类型标识/文献载体标识]. (更新或修改日期)[引用日期]. 获取和访问路径.
```

**示例**：
```
[21] 萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15]. http://www.creader.com/news/20011219/100105001.html.
[22] 李强. 数字化期刊的知识产权问题[J/OL]. 情报学报, 2000, 19(4): 34-38[2001-07-08]. http://periodical.wanfangdata.com.cn/periodical/qbxb/qbxb2000/qbxb0004/000407.htm.
[23] 中国互联网络信息中心. 中国互联网络发展状况统计报告[EB/OL]. (2024-01-30)[2024-02-15]. https://www.cnnic.cn/n4/2022/0708/c88-10226.html.
[24] HOPKINSON A. UNIMARC and metadata: Dublin Core[EB/OL]. (2009-04-22)[2013-03-27]. http://archive.ifla.org/IV/ifla64/138-161e.htm.
```

**著录要点**：
1. 文献载体标识：OL(联机网络)、CD(光盘)、DK(磁盘)、MT(磁带)
2. 更新或修改日期为文献的发布或最后修改日期
3. 引用日期为用户查阅日期，必著录
4. 获取和访问路径为完整的 URL

## 三、著录细则

### 3.1 个人作者著录规则

**中文作者**：
- 姓名按原样著录
- 多作者间用逗号分隔
- 不超过 3 人时全部著录
- 超过 3 人时著录前 3 人后加「, 等」

**示例**：
```
张三, 李四, 王五.
张三, 李四, 王五, 等.
```

**欧美作者**：
- 姓全大写，名缩写
- 姓在前，名在后
- 多作者间用逗号分隔
- 超过 3 人时著录前 3 人后加「, et al.」

**示例**：
```
EINSTEIN A.
EINSTEIN A, BOHR N, PLANCK M, et al.
O'CONNOR J, ROBERTSON E.
```

### 3.2 题名著录规则

1. 题名按文献原语种著录，不翻译
2. 题名中的标点符号保留
3. 副题名用冒号分隔著录
4. 并列题名不著录(除必要时)

**示例**：
```
图书馆目录: 理论与实践
大数据时代的精准营销研究
```

### 3.3 版本项著录规则

| 版本 | 著录形式 |
|------|---------|
| 第1版 | 不著录 |
| 第2版 | 2版 |
| 第3版 | 3版 |
| 新1版 | 新1版 |
| 修订版 | 修订版 |
| 增订版 | 增订版 |

### 3.4 出版项著录规则

**出版地**：
- 著录出版者所在城市名
- 省会城市可省略省份
- 非省会城市需著录省份
- 外国城市可著录城市名或中译名

**出版者**：
- 著录出版机构名称
- 可使用规范简称
- 无出版者时著录「[出版者不详]」

**出版年**：
- 著录公元纪年
- 版权年著录为「c年份」
- 印刷年不作为出版年

### 3.5 页码著录规则

1. 引用单页：著录该页号
2. 引用连续多页：著录起止页号，用连字符
3. 引用不连续多页：著录各页号，用逗号分隔

**示例**：
```
: 15.           # 单页
: 15-20.        # 连续页
: 15, 20, 25.   # 不连续页
: 15-20, 25-30. # 多个连续段
```

## 四、参考文献表排列规则

### 4.1 文献排序方式

**方式一：按正文中引用顺序排序(推荐)**
- 参考文献按正文首次引用顺序编号
- 同一文献多次引用使用相同编号
- 多次引用不同页码时在正文引文处标注页码

**方式二：按作者姓名拼音排序**
- 适用于无引用编号的参考文献表
- 欧美作者按字母顺序排列
- 中文作者按拼音顺序排列

### 4.2 编号格式

- 使用方括号：[1], [2], [3]
- 数字用阿拉伯数字
- 编号与文献间空一格

## 五、正文引用标注规则

### 5.1 标注位置

引用标注置于所引内容最末句的右上角。

**示例**：
> 大数据技术的发展为精准营销提供了新的可能性<sup>[1]</sup>。

### 5.2 标注方式

| 引用情况 | 标注方式 | 示例 |
|---------|---------|------|
| 单篇文献 | [序号] | [1] |
| 多篇文献 | [序号, 序号] | [1, 3] |
| 连续序号 | [序号-序号] | [A-B] |
| 指定页码 | [序号](页码) | [1](15-20) |

### 5.3 多次引用同一文献

同一文献在正文中多次引用时：
- 使用相同的序号
- 在各引用处分别标注页码

**示例**：
> ……这一观点已被证实<sup>[3](15)</sup>。后续研究进一步验证了该结论<sup>[3](20-22)</sup>。

## 六、常见错误及纠正

### 6.1 格式错误

| 错误示例 | 正确示例 |
|---------|---------|
| [1] 张三. 《论文标题》[J]. 期刊名, 2023, 1(1): 1-5. | [1] 张三. 论文标题[J]. 期刊名, 2023, 1(1): 1-5. |
| [2] Li M, Wang Q. ... | [2] LI M, WANG Q. ... |
| [3] 作者. 题名. 期刊名, 2023. | [3] 作者. 题名[J]. 期刊名, 2023, 卷(期): 页码. |
| [4] 题名[J]. 期刊名. 2023;1(1):1-5. | [4] 题名[J]. 期刊名, 2023, 1(1): 1-5. |

### 6.2 常见遗漏

1. ❌ 缺少文献类型标识([J]、[M]等)
2. ❌ 缺少期刊的卷期号
3. ❌ 缺少电子文献的引用日期
4. ❌ 缺少专著的出版地
5. ❌ 页码格式不规范

## 八、使用检查清单

编写参考文献时，对照以下清单逐项检查：

### 期刊论文检查清单
- [ ] 作者姓名格式正确(欧美作者姓全大写)
- [ ] 题名无书名号
- [ ] 文献类型标识 [J] 存在
- [ ] 刊名完整无缩写
- [ ] 年, 卷(期)格式正确
- [ ] 页码格式正确(起止页用连字符)
- [ ] 电子期刊有引用日期和 URL

### 专著检查清单
- [ ] 作者姓名格式正确
- [ ] 题名无书名号
- [ ] 文献类型标识 [M] 存在
- [ ] 出版地存在
- [ ] 出版者存在
- [ ] 出版年存在
- [ ] 版本项(非第1版时)
- [ ] 引用页码存在

### 学位论文检查清单
- [ ] 作者姓名格式正确
- [ ] 题名无书名号
- [ ] 文献类型标识 [D] 存在
- [ ] 保存地存在
- [ ] 保存单位存在
- [ ] 年份存在

### 电子文献检查清单
- [ ] 文献类型/载体标识正确(/OL)
- [ ] 更新或发布日期存在
- [ ] 引用日期存在(必著录)
- [ ] URL 完整可访问
