# 论文降重提示词指南

> 本文档用于指导 AI 进行语义不变的深度改写，降低文本相似度

---

## 一、降重策略总览

### 1.1 策略矩阵

| 策略类型 | 具体操作 | 预估降重效果 | 风险等级 |
|---------|---------|-------------|---------|
| 句式重构 | 主被动转换 | 高 | 低 |
| 句式重构 | 长句拆短句 | 中 | 低 |
| 句式重构 | 短句合长句 | 中 | 中 |
| 同义替换 | 词汇级替换 | 中 | 低 |
| 同义替换 | 短语级替换 | 中 | 低 |
| 段落重组 | 逻辑顺序调整 | 高 | 中 |
| 数据增强 | 具象化描述 | 低 | 低 |
| 引用编织 | 添加引用说明 | 中 | 低 |

### 1.2 执行优先级

```
P0（必做）：句式重构 + 同义替换
P1（建议）：段落重组
P2（可选）：数据增强 + 引用编织
```

## 二、句式重构技术

### 2.1 主被动转换

**示例**：

| 原句 | 改写 |
|-----|-----|
| 本文采用了问卷调查法收集数据。 | 问卷调查法被本研究所采用，以完成数据收集工作。 |
| 企业通过大数据技术实现了精准营销。 | 大数据技术的运用，使精准营销在企业中得以实现。 |
| 研究者对样本进行了分析。 | 样本被纳入分析范围，由研究者完成数据处理。 |

**注意事项**：
- 术语部分不做主被动转换
- 保持学术语体，避免口语化

### 2.2 长句拆短句

**示例**：

| 原句 | 改写 |
|-----|-----|
| 随着互联网技术的快速发展，企业获取消费者数据的渠道日益多元化，这为精准营销的实施提供了重要的数据基础。 | 互联网技术的快速发展改变了数据获取格局。企业获取消费者数据的渠道日益多元化。这一变化为精准营销的实施奠定了数据基础。 |
| 研究发现，消费者购买决策受到多种因素影响，包括产品质量、价格水平、品牌形象以及口碑评价等。 | 研究揭示了消费者购买决策的影响因素。这些因素涵盖多个维度：产品质量、价格水平、品牌形象以及口碑评价。 |

**拆分原则**：
- 一个长句拆成 2-3 个短句
- 每个短句表达一个完整意思
- 短句之间用句号分隔，保持语义连贯

### 2.3 短句合长句

**示例**：

| 原句 | 改写 |
|-----|-----|
| 大数据技术发展迅速。应用领域不断扩展。精准营销成为重要应用场景。 | 在大数据技术迅猛发展、应用领域不断扩展的背景下，精准营销逐渐成为其重要的应用场景之一。 |
| 消费者需求日益个性化。传统营销模式难以满足。企业急需转型。 | 面对消费者日益个性化的需求，传统营销模式已难以适应，企业转型势在必行。 |

**合并原则**：
- 使用关联词（随着、在…背景下、面对…）
- 添加从句结构
- 保持逻辑清晰

### 2.4 句式变换综合示例

**原文**：
> 大数据技术通过对海量消费者数据的收集、存储和分析，帮助企业精准识别目标客户群体，从而制定更有针对性的营销策略，提高营销效率。

**改写方案 1（拆分+主被动）**：
> 海量消费者数据的收集、存储与分析，构成了大数据技术的核心功能。目标客户群体的精准识别由此得以实现。企业据此制定的营销策略更具针对性，营销效率随之提升。

**改写方案 2（合并+重排）**：
> 为提升营销效率，企业借助大数据技术完成三项核心工作：收集存储海量消费者数据、精准识别目标客户、制定针对性营销策略。

## 三、同义替换技术

### 3.1 词汇级替换

**常用同义词对照表**：

| 原词 | 替换选项 |
|-----|---------|
| 显著 | 明显、突出、可观、引人注目 |
| 提高 | 提升、增进、加强、改善 |
| 问题 | 难题、困境、挑战、议题 |
| 方法 | 方式、途径、手段、路径 |
| 影响 | 作用、效应、冲击、波及 |
| 发展 | 进步、演进、成长、推进 |
| 重要 | 关键、核心、要紧、举足轻重 |
| 实施 | 开展、推行、落实、执行 |
| 分析 | 剖析、研究、探究、考察 |
| 结果 | 成果、产出、结论、成效 |

### 3.2 短语级替换

| 原短语 | 替换选项 |
|-------|---------|
| 起到了重要作用 | 发挥了关键效能 / 具有举足轻重的地位 |
| 随着科技的发展 | 在技术进步的推动下 / 伴随技术革新 |
| 对…进行了研究 | 针对…开展了探究 / 围绕…实施了调研 |
| 取得了显著成效 | 收获了可观成果 / 达成了预期目标 |
| 在一定程度上 | 某种意义上 / 就部分而言 |
| 由此可见 | 这意味着 / 这说明 / 不难看出 |
| 与此同时 | 同期 / 同一时间 / 在此期间 |

### 3.3 Python 辅助替换

使用 `scripts/synonym_replace.py` 进行自动同义词替换：

```powershell
# 基本替换（默认 30% 替换率）
python scripts/synonym_replace.py --input paper.md --output paper_replaced.md

# 自定义替换比例
python scripts/synonym_replace.py --input paper.md --ratio 0.4

# 指定术语白名单（保护专业术语不替换）
python scripts/synonym_replace.py --input paper.md --whitelist scripts/term_whitelist.txt
```

## 四、段落重组技术

### 4.1 逻辑顺序调整

**因果倒置**：

| 原文（原因→结果） | 改写（结果→原因） |
|-----------------|-----------------|
| 由于市场竞争加剧，企业不得不寻求新的营销方式。 | 企业寻求新的营销方式，这一转变源于市场竞争的日益加剧。 |

**现象本质互换**：

| 原文（现象→本质） | 改写（本质→现象） |
|-----------------|-----------------|
| 消费者购买行为日益理性化，这反映了市场成熟度的提升。 | 市场成熟度的提升，在消费者购买行为理性化趋势中得以体现。 |

### 4.2 论证顺序重组

**原文**：
> 精准营销的核心在于数据。首先，数据为消费者画像提供基础。其次，数据支撑营销决策。最后，数据用于效果评估。

**改写 1（倒序）**：
> 数据之于精准营销，贯穿全流程：效果评估依赖数据反馈，决策制定基于数据分析，消费者画像源于数据采集。三者共同构成精准营销的数据闭环。

**改写 2（归纳式）**：
> 精准营销的数据应用涵盖三个关键环节：消费者画像构建、营销决策支撑、效果评估反馈。这三个环节环环相扣，共同支撑精准营销的实施。

## 五、数据增强与引用编织

### 5.1 数据具象化

| 原文（抽象） | 改写（具象） |
|------------|------------|
| 显著提升 | 提升了 23.7% |
| 大幅增长 | 从 120 万增至 185 万，增幅达 54.2% |
| 占据主导地位 | 市场份额达 67.3% |
| 多数企业 | 调查显示，78.5% 的受访企业 |

**注意事项**：
- 数据需合理可信
- 可注明「据估算」「研究显示」
- 避免过于精确引发质疑

### 5.2 引用编织

**原文**：
> 大数据技术对精准营销具有重要意义。

**改写**：
> 大数据技术对精准营销的价值已得到学界广泛认可。张三（2023）指出，大数据技术使精准营销效率提升了 30% 以上；李四（2022）的研究进一步证实，应用大数据技术的企业营销 ROI 平均提高 2.5 倍。

## 六、降重流程

### 6.1 标准流程

```
1. 阅读原文，理解核心观点
2. 标记关键术语（不可替换）
3. 执行句式重构（主被动、拆分、合并）
4. 执行同义替换（词汇、短语）
5. 检查段落逻辑，必要时重组
6. 朗读检查，确保语句通顺
7. 保存改写版本，记录改动
```

### 6.2 改写强度控制

| 强度 | 适用场景 | 操作特点 |
|------|---------|---------|
| 轻度 | 原文查重率 20-30% | 仅做句式微调 |
| 中度 | 原文查重率 30-50% | 句式重构 + 同义替换 |
| 重度 | 原文查重率 > 50% | 全面改写，包括段落重组 |

### 6.3 版本管理

每次降重前，自动备份原文件到 `workspace/history/`：

```
workspace/history/
├── chapter_01_v1_20260305_1430.md  # 第一轮降重前
├── chapter_01_v2_20260305_1455.md  # 第二轮降重前
└── ...
```
